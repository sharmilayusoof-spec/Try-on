COMMON ISSUES & QUICK FIXES
================================================================================

Issue: "Cannot connect to server"
Fix: Start backend → uvicorn app.main:app --reload

Issue: Cloth images not showing
Fix: Hard refresh → Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

Issue: Processing forever
Fix: Check backend terminal - where does it stop?

Issue: No result shown
Fix: Run in console → document.getElementById('resultSection').style.display = 'block'
