Manual Start (All Platforms)

Open Terminal/Command Prompt

Navigate to your project root directory (where the app/ folder is located).

Run This Command

uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

Wait for Startup

You should see: INFO: Application startup complete.

Test Backend

Open browser and go to: http://localhost:8000/docs

You should see the API documentation page.

Refresh Frontend

Go back to your frontend page and try again!

‚ö†Ô∏è Common Issues

  • "uvicorn: command not found" ‚Üí Install it: pip install uvicorn
  • "No module named 'app'" ‚Üí Make sure you're in project root, not inside app/ folder
  • "Port 8000 already in use" ‚Üí Kill the process or use different port
  • Missing dependencies ‚Üí Install: pip install -r requirements.txt
