================================================================================
TRY-ON PROCESSING BUT NO RESULT - QUICK FIX
================================================================================

PROBLEM: After clicking "Try On", button shows "Processing..." but no result
         is displayed after processing completes.

================================================================================
QUICK DIAGNOSIS (Choose One)
================================================================================

Option 1: Diagnostic Script (Recommended)
  1. Open app in browser
  2. Press F12 → Console tab
  3. Copy entire contents of: diagnose_tryon_result.js
  4. Paste into console and press Enter
  5. Click "Try On" button
  6. Watch detailed logs

Option 2: Test Page
  1. Open in browser: test_tryon_result.html
  2. Upload user and cloth images
  3. Click "Test Try-On API"
  4. See step-by-step results

Option 3: Manual Check
  1. Press F12 → Console tab
  2. Click "Try On"
  3. Look for errors (red text)
  4. Check Network tab for /tryon/process request

================================================================================
MOST COMMON CAUSES
================================================================================

1. Backend not returning image URL
   → Check response has 'image_url' or 'url' field

2. Result section not showing
   → Should have display: block after processing

3. Image file not saved
   → Check results/ directory exists and has files

4. Static files not served
   → Verify app.mount("/results", ...) in main.py

5. displayResult() not called
   → Verify processTryOn() calls displayResult()

================================================================================
EMERGENCY FIX
================================================================================

Run in browser console (F12):

// Force show result section
const resultSection = document.getElementById('resultSection');
resultSection.style.display = 'block';
resultSection.scrollIntoView({ behavior: 'smooth' });

// Check if result section exists
console.log('Result section:', {
    exists: !!resultSection,
    display: resultSection?.style.display,
    visible: resultSection?.offsetHeight > 0
});
